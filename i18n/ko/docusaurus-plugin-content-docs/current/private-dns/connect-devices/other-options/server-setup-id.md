## 서버 Setup ID란 무엇인가요?

특정 서버에 기기를 연결하려는데 웹사이트의 대시보드에서 각 기기를 수동으로 생성하고 싶지 않다면, **서버 Setup ID**를 사용할 수 있습니다. ID를 가져와 AdGuard DNS 앱을 통해 연결하세요. 이렇게 하면 연결 시 '기기'가 자동으로 생성됩니다.

## 서버 Setup ID를 사용하여 연결하는 방법

1. **대시보드** → [**서버**](https://adguard-dns.io/dashboard/servers)로 이동합니다.
2. 원하는 서버를 선택합니다.
3. 아래로 스크롤하여 **Setup ID**를 클릭합니다.
   ![Setup ID \*mobile_border](https://cdn.adtidy.org/content/release_notes/dns/dns-app-v1-0/server_settings_setup_ID.png)
4. **Setup ID**를 복사합니다.
   ![서버 Setup ID \*mobile_border](https://cdn.adtidy.org/content/release_notes/dns/dns-app-v1-0/settings_setup_ID.png)
5. 앱을 열고 **Setup ID**를 붙여넣습니다.
   ![Setup ID 붙여넣기 \*mobile_border](https://cdn.adtidy.org/content/release_notes/dns/dns-app-v1-0/SetupIDMobile.png)
6. 기기 이름을 선택합니다.
7. **연결**을 누릅니다.
   ![연결 \*mobile_border](https://cdn.adtidy.org/content/release_notes/dns/dns-app-v1-0/ConnectMobile.png)

이제 연결이 완료되었으며 대시보드에 기기가 자동으로 생성됩니다.

## 서버 Setup ID 초기화

특정 Setup ID를 사용 중인 모든 기기의 연결을 끊으려면 해당 ID를 초기화할 수 있습니다. Setup ID를 초기화하려면 다음 단계를 따르세요.

1. **대시보드** → [**서버**](https://adguard-dns.io/dashboard/servers)로 이동하여 원하는 서버를 선택합니다.
2. 아래로 스크롤하여 **Setup ID**를 클릭합니다.
3. **Setup ID 초기화**을 클릭합니다.

초기화 후 설정 ID가 변경되며 이전 ID를 사용하던 모든 기기의 서버 연결이 끊어집니다.
