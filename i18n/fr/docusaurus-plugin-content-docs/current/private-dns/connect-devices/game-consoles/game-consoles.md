---
title: Consoles de jeux
sidebar_position: 1
---

Les consoles de jeux ne prennent pas en charge le DNS crypté, mais elles sont bien adaptées à la configuration de l'AdGuard DNS public ou de l'AdGuard DNS privé via une adresse IP liée.

- [Nintendo](private-dns/connect-devices/game-consoles/nintendo.md)
- [Nintendo Switch](private-dns/connect-devices/game-consoles/nintendo-switch.md)
- [PlayStation](private-dns/connect-devices/game-consoles/playstation.md)
- [Steam Deck](/private-dns/connect-devices/game-consoles/steam.md)
- [Xbox One](private-dns/connect-devices/game-consoles/xbox-one.md)
